
**SYNASTER** — это мощный инструмент для массовой рассылки сообщений в Telegram. Программа позволяет отправлять сообщения в чаты, группы и каналы через ваш аккаунт Telegram.

## 🚀 Возможности

- 📨 **Массовая рассылка** сообщений в Telegram
- 👥 **Поддержка различных типов целей**:
  - Username (`@channel_name`)
  - Ссылки (`https://t.me/group_name`)
  - ID чатов (`-1001234567890`)
- ⚡ **Асинхронная отправка** для высокой скорости
- 📊 **Статистика** отправки (успешно/ошибки)
- 🔄 **Автоматическая обработка** FloodWait ошибок
- 💾 **Сохранение конфигурации** API данных
- ⌨️ **Удобное меню** с навигацией

## 📋 Требования

- Python 3.8 или выше
- Установленные зависимости (см. ниже)
- API ID и API Hash от [my.telegram.org](https://my.telegram.org)
- Активный аккаунт Telegram
